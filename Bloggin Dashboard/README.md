# Bloggin Dashboard Project
Bootstrap Dashboard Project for managing Blogs

# Technologies Used
- HTML
- CSS
- SCSS
- Bootstrap

# Project Specifications 
- Basic Web Template using Boostrap 4.0
- Smooth Scrolling
- Fully Responsive
- Multi-Page & Multi-Function Blogging Dashboard Website
    - Dashboard
    - Posts
    - Categories
    - Users
    - Settings
- Search Box
- Login
- Adding Users
- Categories
- Save/Delete Blogs
- Profile Page
- Change Password
